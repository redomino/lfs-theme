Introducion
===========

This is a responsive theme for LFS based on twitter bootstrap.
LFS: http://getlfs.com
twitter bootstrap: http://twitter.github.io/bootstrap/

Now we have:
 - responsive two/three columns layout
 - only bootstrap css are active
 - fluid category view showing all products or a table of them (depending on row and column properties).
 - product view is now responsive
 - static homepage is responsive
 - applied bootstrap style at many elements like: main navigation, left navigation, search and others buttons.
 - main navigation now shows only first level categories
